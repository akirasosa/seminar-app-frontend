VPC Hands-on Help Page

This page helps you to create your first VPC.

example.tf

This is all what you need to create your first VPC.

resource "aws_vpc" "main" {
  cidr_block = "${var.vpc_cidr}"
}

provider "aws" {
  access_key = "${var.aws_access_key}"
  secret_key = "${var.aws_secret_key}"
  region = "${var.aws_region}"
}

variable "aws_access_key" {
  default = "{{setting.accessKey}}"
}
variable "aws_secret_key" {
  default = "{{setting.secretKey}}"
}
variable "aws_region" {
  default = "{{setting.region}}"
}
variable "vpc_cidr" {
  default = "{{setting.vpcCidr}}"
}

Info Though I wrote it as a single file, it would be a good idea to separate it. For example...

project_dir/
  vpc.tf          # define your vpc
  variables.tf    # define your variables
  aws.tf          # define your aws provider settings

Steps to create your first VPC

  1. Download Terraform from download page.
  2. Put the binary into somewhere in your PATH.
  3. Save example.tf into your project dir.
  4. Run terraform plan in your project dir.
  5. Run terraform apply.
  6. Confirm that your VPC was created on AWS Console.

You can sign in to AWS console. Account ID is {{setting.username}}. Password is {{setting.password}}. Don't forget to move to your region {{setting.region}} after visiting EC2 or VPC page.
